This is a small model — responses may be inconsistent or nonsensical. Try the 2D Classifier and Token Explorer tabs to see how the model makes its predictions.

Peek behind the curtain. This tab uses a logistic regression model (just 6 parameters: 2 weights + 1 bias per class) to classify 2D points into two categories. The model receives an (x, y) coordinate, produces a raw score (logit) for each class, then converts those scores into probabilities using softmax. The dashed line shows the decision boundary — where the model is exactly 50/50.

The model starts with random weights, so the boundary is arbitrary — click Train Model to fit it to the data (instant, 200 steps of SGD) and watch the boundary snap into place. Then try the Token Explorer tab to see the same process applied to text, with 50,000+ possible tokens instead of 2.
