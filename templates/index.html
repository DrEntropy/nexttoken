This is a small model — responses may be inconsistent or nonsensical. Try the Digit Classifier and Token Explorer tabs to see how the model makes its predictions.

Now peek behind the curtain. This tab uses a small neural network (~60,000 parameters) to classify hand-drawn digits. The model receives your drawing, produces a raw score (logit) for each possible answer, then converts those scores into probabilities (using a function called softmax). With only 10 possible answers (digits 0–9), you can see the full distribution at a glance.

The model starts with random weights, so predictions are initially random — click Train Model to train it on MNIST (a classic dataset of 70,000 handwritten digits) and watch predictions become confident and accurate. Then try the Token Explorer tab to see the same process applied to text, with 50,000+ possible tokens instead of 10.
